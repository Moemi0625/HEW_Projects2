Polyglot Memory Quest : Chasing Translation

How To Play

"Polyglot Memory Quest : Chasing Translation" is a new and exciting memory card game where you can enjoy learning languages and cultures by matching flags, greetings, and cultural image cards which express each different countries. Your goal is to correctly match all combinations and translate into all languages! There are a total of 3 countries in the game – can you guess which ones?

  • Click START button to begin the game, and RESET
    button to reset and restart it.
  • If you are unsure, check hints from HINT button.
  • There is no time limit, so take your time to think.
  • After you complete the game, choose the language
    from the options will be given.

Enjoy the adventure of discovering new words and cultures!

Polyglot Memory Quest：翻訳を目指して

ゲームのあそび方について

「多言語記憶の探求：翻訳を目指して」は、言語と文化を楽しみながら学ぶ新感覚のメモリーカードゲームです。 各国の国旗、あいさつ、文化を表すカードを国ごとに正しくマッチングしていきます。 目指すはすべてのカードの組み合わせを見つけ出し、サイトの記事を他国の言語に翻訳すること！ ゲームに出てくる国は全部で3か国。さて、一体どこの国か、推測できるでしょうか？

  • STARTボタンでゲーム開始、RESETボタンで
    ゲームリセットです。
  • どうしてもわからない時はHINTボタンからヒントを
    確認できます。
  • 時間制限はありません、じっくり考えてください。
  • ゲーム終了時に、翻訳する言語を選択肢から選んでください。

では、新しい言葉と文化の冒険をぜひ楽しんでください♪
